Склад, ревизия
Посмотреть видеовыбрать тариф
Склад, ревизия
  • Учет наличия в автосервисе, с возможностью добавить товар в заказ наряд;
  • Настройка минимальных остатков на складе, программа отправит товары на закупку
  • Рассылка прайсов клиентам
  • Перемещение товаров внутри склада при помощи мобильно приложения
  • Ревизия в на складе автосервиса с мобильным приложением
  • Стикерование товаров с указанием полки, где будет лежать товар
  • Автоматическое создание штрихкодов, артикулов
  • Комплектование товаров
Поиск запчастей в автосервисе
Посмотреть видеовыбрать тариф
Поиск запчастей в автосервисе
  • Встроенные автокаталоги;
  • Поиск запчастей по VIN/Frame коду клиента
  • Большое количество веб-сервисов различных поставщиков
  • Готовая база кроссов, а также возможность включить опцию «Доверенный поставщик по кроссам»
  • Подбор запчастей по файлу клиента
  • Настройка сроков для поставщиков, изменение отображения цветов по каждому складу поставщика
  • Кроссирование вашего наличия при поиске запчастей у посатвщиков
Клиенты, гараж клиентов, история

Клиенты, гараж клиентов, история
  • Ведение клиентской базы, гараж клиента;
  • История заказов запчастей по клиенту и по каждому автомобилю клиента
  • Добавление различных уровней цен для клиентов
  • Прием предоплаты по заказам, авансовые платежи от клиентов
  • Рассылка своего прайса по почте, веб-сервису
  • Программа лояльности для клиентов(Процентные карты, бонусы)
  • Продажа/отгрузка товаров в кредит
  • Выставление счетов и отправки их по почте из программы
  • Программа лояльности для клиентов(Процентные карты, бонусы)
  • Продажа/отгрузка товаров в кредит
  • Выставление счетов и отправки их по почте из программы
Заказ-наряды, запись по графику в сервис
Посмотреть видеовыбрать тариф
Заказ-наряды, запись по графику в сервис
  • Запись клиентов по графику, с возможностью показа записей на всю неделю;
  • Заказ-наряды, счета для организаций;
  • Смс оповещение клиентов о готовности автомобиля, о бронировании, напоминание за день до брони и т.д
  • База нормачасов
  • Базовый список услуг по узлам и усредненным нормочасам;
  • Возможность записать клиента на ремонт через мобильное приложение
  • История работ и заказов запчастей по автомобилю и клиенту
  • Добавление фото к заказ наряду
Мастера, мастера приемщики и зарплата
Посмотреть видеовыбрать тариф
Мастера, мастера приемщики и зарплата
  • Возможность указывать процент выполнения для каждого мастера;
  • Расчет заработной платы согласно выполненным работам;
  • Отдельный план для мастера приемщика с возможностью настройки плана по продаже запчастей
  • Начисление зарплаты согласно выработке по нормачасам
  • Назначение мастеров на различные работы
  • Ограничение полномочий сотрудников внутри программы. Запрет провести заказ наряд задним числом, перепровести его, удалить или изменить.
Розница, касса и терминалы
Посмотреть видеовыбрать тариф
Розница, касса и терминалы
  • Различные печатные формы (акт выполненных работ, заказ наряд, карточка осмотра и т.д.);
  • Отчет по выработке каждого мастера с/без учетом нормачасов
  • Отчет по маржинальности/прибыльности каждого заказ наряда
  • Готовые печатные формы заказ-наряда, акт выполненных работ, акт диффектовки и т.д.
  • Возможность отредактировать любую печатную форму под свой стиль

  • Быстрая реализация товаров через Окно кассира
